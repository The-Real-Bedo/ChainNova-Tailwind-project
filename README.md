# 🚀 ChainNova - Tailwind CSS Practice Project

A modern, responsive landing page for a fictional blockchain company built as a practical project while learning **Tailwind CSS**.

The goal of this project was to practice building real-world responsive layouts, reusable utility classes, and modern UI components using Tailwind CSS.
